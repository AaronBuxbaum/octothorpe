## Install
  * Database: `brew install mongodb`
  * Flask: `brew install python && pip install -r server/requirements.txt`
  * Client: `yarn`

## Start
  * Database: `yarn database`
  * Server: `yarn server`
  * Flask: `yarn flask`
  * Client: `yarn start`

## Docker
  * Install Docker
  * Log In: `docker login`
  * To build: `docker-compose build`
  * To start: `docker-compose up`
